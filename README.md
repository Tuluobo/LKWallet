## 编译

使用最新的 Cocoapods（version 1.3.1）和 Xcode 9.2 进行编译并运行。

- 首先，下载源代码：
  `git clone https://github.com/Tuluobo/LKWallet.git`

- 使用终端切换到项目目路，执行：
  `pod install`

  > 如果没有安装 `CocoaPods` 请 Google 安装。


- 打开 `LKWallet.xcworkspace` Build, Run。
